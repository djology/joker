## 目录操作
```
ls（list）
    功能描述：显示目录文件`  
    常用选项：
            -a，显示隐藏文件
            -l，纵向显示文件信息
            -h，人性化显示文件大小
            -d，显示目录本身，而不是里面的子文件
    通配符：
            *，表示任意个数个字符
            ?，表示任意一个字符
            []，表示可以匹配字符组中任意一个
            [abc]，匹配a，b，c中任意一个
            [a-f]，匹配a到f范围内任意一个

cd（change directory）
    功能描述：切换所在路径
    常用选项：
            cd ~，进入当前用户的家目录（/home/用户目录）
            cd -，进入上次目录
            cd ..，回到上级目录

pwd（print working directory）
    功能描述：显示当前工作目录

mkdir（make directory）
    功能描述：创建目录
    常用选项：
            -p，递归创建

rmdir（remove empty directory）
    功能描述：删除目录（只能删除空目录）
```
## 文件操作
```
touch
    功能描述：创建空文件 或 更新文件修改时间（如果文件已存在）

rm（remove）
    功能描述：删除
    常用选项：
            -r，递归删除文件和目录
            -f，强制删除该目录，不提示

tree
    功能描述：以树状图列出文件目录结构
    常用选项：
            -d，只显示目录

cp（copy）
    功能描述：复制文件或目录
    常用选项：
            -f，目标文件已存在则会直接覆盖，不提示
            -i，覆盖前提示
            -r，源文件是目录文件，直接cp；源文件是子目录和文件，目标文件须是目录名
            -p，
            -d，

mv（move）
    功能描述：移动文件或目录，也可以对文件或目录重命名
    常用选项：
            -i，覆盖前提示
```
## 查看文件内容
```
cat（concatenate）
    功能描述：查看文件内容
    常用选项：
            -b，对非空输出行编号
            -n，对输出所有行编号

more
    功能描述：分屏显示文件内容
    快捷键：
            向上翻页：空格键
            向下翻页：b
            退出查看：q

grep
    功能描述：对文本进行模式查找
    常用选项：
            -n，显示匹配行及行号
            -v，显示未匹配行的所有行
            -i，忽略大小写
    常用的模式查找：
            ^a，行首
            ke$，行尾

echo
    功能描述：在终端显示参数指定内容
            
head
    功能描述：显示文件头（默认10行）
    常用选项：
            -n，可以限制显示行数
            
ln（link）
    功能描述：链接文件，相当于快捷方式
    常用选项：
            -s，
```
