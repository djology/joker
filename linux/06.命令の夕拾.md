## 目录操作
```
ls（list）
    功能描述：显示目录文件`  
    常用选项：
            -a，显示隐藏文件
            -l，纵向显示文件信息
            -h，人性化显示文件大小
            -d，显示目录本身，而不是里面的子文件
    通配符：
            *，表示任意个数个字符
            ?，表示任意一个字符
            []，表示可以匹配字符组中任意一个
            [abc]，匹配a，b，c中任意一个
            [a-f]，匹配a到f范围内任意一个

cd（change directory）
    功能描述：切换所在路径
    常用选项：
            cd ~，进入当前用户的家目录（/home/用户目录）
            cd -，进入上次目录
            cd ..，回到上级目录

pwd（print working directory）
    功能描述：显示当前工作目录

mkdir（make directory）
    功能描述：创建目录
    常用选项：
            -p，递归创建

rmdir（remove empty directory）
    功能描述：删除目录（只能删除空目录）
```
## 文件操作
```
touch
    功能描述：创建空文件 或 更新文件修改时间（如果文件已存在）

rm（remove）
    功能描述：删除
    常用选项：
            -r，递归删除文件和目录
            -f，强制删除该目录，不提示

tree
    功能描述：以树状图列出文件目录结构
    常用选项：
            -d，只显示目录

cp（copy）
    功能描述：复制文件或目录
    常用选项：
            -f，目标文件已存在则会直接覆盖，不提示
            -i，覆盖前提示
            -r，源文件是目录文件，直接cp；源文件是子目录和文件，目标文件须是目录名
            -p，
            -d，
            -a，

mv（move）
    功能描述：移动文件或目录，也可以对文件或目录重命名
    常用选项：
            -i，覆盖前提示
```

## 远程管理
```
shutdown
    功能描述：安全关闭或重启系统
    常用选项：
            -r，重新启动
    提示：
            不指定选项和参数，默认1分钟后关闭系统
    示例：
            # 立即重启操作系统
            $ shutdown -r now
            
            # 立即关机
            $ shutdown now
            
            # 系统将在今天的20:25关机
            $ shutdown 20:25
            
            # 系统再过10分钟关机
            $ shutdown +10
            
            # 取消之前设定的关机时间
            $ shutdown -c

ifconfig（configure network interface）
    功能描述：查看网卡配置信息
    示例：
            # 查看网卡对应的IP地址
            $ ifconfig | grep inet

ping
    功能描述：检测网络是否正常连接

ssh（secure shell）
    功能描述：是专为远程登录会话和其他网络服务提供安全性的协议
    端口号：（可以通过端口号找到计算机上运行的应用程序）
            SSH服务器：22
            Web服务器：80
            HTTPS：443
            FTP服务器：21
    示例：
            ssh [-p port] user@remote
            
                user，远程机器上运行的用户
                remote，远程机器的地址，可以是IP/域名，也可以是别名
                port，SSH Server监听的端口，默认为22
    引深：
            有关SSH配置信息都保存在用户家目录下的.ssh目录下
            免密码登录配置
                执行ssh-keygen生成SSH钥匙（在.ssh目录下配置）
                执行ssh-copy-id -p port user@remote，以让远程服务器记住密钥
                
                涉及非对称加密算法：
                    使用公钥加密的数据，需要使用私钥解密
                    使用私钥加密的数据，需要使用公钥解密
                
            别名配置
                在~/.ssh/config追加（gedit config）以下内容，可以略写成ssh mac
                    Host mac
                        HostName ip地址
                        User djology
                        Port 22

scp（secure copy）
    功能描述：远程复制文件
    常用选项：
            -r，递归复制子目录及文件
            -P，指定端口
```
## 用户权限
```
在Linux系统中，不同的系统都必须拥有一个用户，且对不同的系统资源拥有不同的使用权限
对 文件/目录 的权限包括：
        读       read        r       4
        写       write       w       2
        执行     excute      x       1
        无                   _       0

在Linux中，为了方便用户管理，提出 组 的概念，预先设置组的权限，可以将不同的用户添加到对应的组中

ls -l 扩展
    ls -l 可以查看文件下文件的详细信息，从左到右依次是：
            权限
            硬链接数，目录下的子目录越多，硬链接数越多
            用户
            组
            大小
            时间
            名称
    
    权限细分：目录 拥有者权限 组权限 其他用户权限（d表示目录）
```
